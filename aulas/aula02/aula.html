Manipulação de Variáveis e Tipos de Dados

Exemplo de Variáveis:

// Declaração e atribuição de variáveis
var idade = 25;
let nome = "João";
const pi = 3.14;

// Exibindo variáveis no console
console.log("Idade:", idade);
console.log("Nome:", nome);
console.log("Valor de pi:", pi);

Tipos de Dados:

// Tipos de dados em JavaScript
var numeroInteiro = 42;
var numeroDecimal = 3.14;
var texto = "Isso é uma string";
var booleano = true;

// Exibindo tipos de dados no console
console.log("Número Inteiro:", numeroInteiro);
console.log("Número Decimal:", numeroDecimal);
console.log("Texto:", texto);
console.log("Booleano:", booleano);

Operações com Variáveis:

// Operações com variáveis
var x = 10;
var y = 5;

var soma = x + y;
var subtracao = x - y;
var multiplicacao = x * y;
var divisao = x / y;

// Exibindo resultados no console
console.log("Soma:", soma);
console.log("Subtração:", subtracao);
console.log("Multiplicação:", multiplicacao);
console.log("Divisão:", divisao);

Conversão de Dados:

É possível converter dados entre diferentes tipos em JavaScript. Veja alguns exemplos:

// Convertendo para String
var numero = 42;
var numeroString = String(numero);

console.log("Número como String:", numeroString);
console.log("Número como String:", numeroString + 10);

// Convertendo para Número
var textoNumero = "123";
var numeroConvertido = Number(textoNumero);

console.log("Texto para Número:", numeroConvertido);
console.log("Texto para Número:", numeroConvertido + 10);

// Convertendo para Booleano
var valorTruthy = "Olá";
var valorFalsy = "";

console.log("Valor Truthy para Booleano:", Boolean(valorTruthy));
console.log("Valor Falsy para Booleano:", Boolean(valorFalsy));
