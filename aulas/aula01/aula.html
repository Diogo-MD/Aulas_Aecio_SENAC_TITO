Aula 01 - Introdução ao JavaScript

O que é JavaScript?

JavaScript é uma linguagem de programação de alto nível, interpretada, e orientada a objetos. Amplamente utilizada para o desenvolvimento web, ela proporciona interatividade em páginas da web, tornando-as mais dinâmicas e responsivas.

Diferenciais do JavaScript

  • Executado no lado do cliente (navegador), proporcionando uma experiência mais ágil para o usuário.
  • Compatível com várias plataformas e navegadores.
  • Ampla comunidade e rica oferta de bibliotecas (ex: React, Angular, Vue) e frameworks.
  • Pode ser aplicado tanto no desenvolvimento front-end quanto no back-end (Node.js).

Diferentes Formas de Realizar Inputs em JavaScript

1. Prompt: Recebendo dados do usuário

Código:

function promptInput() { var nome = prompt("Digite seu nome:"); console.log("Nome digitado: " + nome); }

2. Confirm: Obtendo confirmação do usuário

Código:

function promptConfirmacaoInput() { var confirmacao = confirm("Você deseja continuar?"); console.log("Confirmação: " + confirmacao); }

3. Input de Texto: Usando elementos de formulário

Código (sem html TAGS):

input type="text" id="campoTexto" placeholder="Digite algo"
button onclick="obterTexto()">Obter Texto button
script
function obterTexto() { var texto = document.getElementById("campoTexto").value; console.log("Texto digitado: " + texto); }
